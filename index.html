Bootcamp Project

Feed-A-Star-Mole Game

This project features from what it seems an easy game, however it took a lot of work. Pausing on a single line of code, fixing bugs and errors while understanding the mechanism behind them. It's great that I could finish it and explore core javascript concepts along the way and also many tricky css concepts ! Project repository.
Rules : When mole looks out of the hole, with open mouth you hover on him and click. By that you fed the mole ! Each mole gets you points. King moles are worth the double and show less often.

Technologies used include:

  • HTML
  • CSS
  • JavaScript
